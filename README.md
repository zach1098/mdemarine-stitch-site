# MDEmarine Stitch Site

Static marketing site rebuilt from the Google Stitch export.

## Files

- `index.html` — the site
- `vercel.json` — simple Vercel static config

## Local preview

Open `index.html` in a browser.

## Deploy to GitHub + Vercel

1. Create a new GitHub repo.
2. Upload these files from this folder.
3. In Vercel, click **Add New Project**.
4. Import the GitHub repo.
5. Framework preset: **Other**.
6. Build command: leave blank.
7. Output directory: leave blank.
8. Deploy.

## Before publishing

Replace the placeholder contact info in the contact section:

- phone number
- email
- service area
- form handler if you want submissions to work
