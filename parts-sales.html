Useful parts-page copy

Tell people what you need from them before chasing stock

Without engine data, serial information, or photos, a parts request becomes a guessing game. This page should do intake work first.

Identification

Get the part right first

Ask for engine model, serial number, and photos before anyone starts naming components off a blurry phone call.

  • Engine tag and serial photo
  • Current component photo from multiple angles
  • Context on symptoms or reason for replacement
Scoping

Repair versus replacement

Sometimes the issue is not just a missing part, it is a larger engine or fabrication problem. The page should acknowledge that early.

  • Confirm whether the failure is already diagnosed
  • Decide if sourcing alone solves the problem
  • Escalate to engine or fabrication scope when needed
Timing

Urgent, planned, or tied to yard dates

Lead time matters. A clean parts page should separate emergency needs from planned jobs so the next step is obvious.

  • Boat down right now
  • Scheduled repair within a known window
  • Repower or fabrication project still in planning
What the request should include

Use this page to collect better parts requests

  • Engine make, model, serial, and horsepower
  • Name or photo of the failed part if known
  • Whether the part is OEM-specific or an acceptable aftermarket replacement
  • Whether install help or fabrication support is also needed
  • Deadline, boat location, and whether the project is already in a yard schedule

This helps visitors send a request someone can actually act on instead of forcing a long back-and-forth before the basics are known.

Best handoffs

Where parts requests usually go next

  • Back to engine services if the failure itself is still unclear
  • To fabrication when the replacement needs custom fitment, mounts, or support pieces
  • To the main request form when the visitor already has enough detail to start the conversation
